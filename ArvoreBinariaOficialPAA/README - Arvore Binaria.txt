INSTRUÇÕES - ÁRVORE BINÁRIA

Como estamos medindo a performance dos algoritmos, dividimos em 3 arquivos:
a) Inserção
b) Remoção
c) Busca

a) Inserção:
1- Ao abrir a inserção, deve-se digitar a quantidade de números a serem gerados
pelo programa.
OBS: Como é necessário que os números não se repitam, e que sejam de 0 até
o numero N desejado, após os números serem gerados, o programa verifica
se há algum número repetido. Então, se colocar um número muito grande (Ex: 50.000),
demorará muito para verificar se há algum repetido gerado.
2- Logo em seguida é exibido o tempo que foi levado para realizar a operação
de inserção

-----------------------
b) Remoção:
1- Ao abrir a remoção, deve-se digitar a quantidade de números a serem gerados
pelo programa (leve em consideração a observação acima citada).
2- Digite o número a ser removido.
3- Se a mensagem: "Valor não encontrado" for exposta, significa que o programa
não encontrou o número desejado. Caso contrário, o valor foi removido
4- É exibido o tempo que foi levado para realizar a operação de remoção.

-----------------------
c) Busca:
1- Ao abrir a busca, deve-se digitar a quantidade de números a serem gerados
pelo programa (leve em consideração a observação acima citada).
2- Digite o número a ser buscado.
3- Se a mensagem: "Valor xxx não encontrado" for exposta, significa que o programa
não encontrou o número desejado. Caso contrário, o valor foi encontrado.
4- É exibido o tempo que foi levado para realizar a operação de busca.
